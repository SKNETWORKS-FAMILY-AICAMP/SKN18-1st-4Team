# SKN18-1st-4Team

## 1. 프로젝트 개요

### 1.1 프로젝트 목적
- 일반 소비자에게 친환경 자동차 구매의 필요성과 장점을 시각적으로 전달
- 친환경 자동차 보조금 현황 및 지원 정보 제공
- 현대자동차와 기아자동차의 FAQ 정보 제공

### 1.2 기술 스택
- **프론트엔드**: Streamlit (Python)
- **데이터베이스**: MySQL (DBeaver 연동)
- **데이터 시각화**: Plotly, Matplotlib
- **프로젝트 구조**: 모듈화된 유틸리티 구조

## 2. 시스템 아키텍처

### 2.1 폴더 구조
```
SKN18-1st-4Team
├── app.py                 # 메인 애플리케이션 (1페이지)
├── money.py          # 보조금 정보 페이지 (2페이지)
├── FAQ.py            # 기업 FAQ 페이지 (3페이지)
├── utilities/
│   ├── app_utility.py     # 1페이지 유틸리티
│   └── money_utility.py   # 2페이지 유틸리티
│   └── faq_utility.py     # faq 페이지 유틸리티
├── database/
│   └── database.py        # 데이터베이스 연결 및 쿼리
└── README.md
```

### 2.2 데이터베이스 구조
- **데이터베이스명**: car
- **주요 테이블**:
  - `electronic_car`: 전기차 정보 (국가보조금, 지자체보조금) 테이블 
  - `hydrogen_car`: 수소차 정보 (지원금) 테이블
  - `environmental_vehicles` : 친환경 자동차 연도별 데이터 테이블 (수소차,전기차,하이브리드, 합계)
  - `greenhouse_gases` - 온실가스 배출량 관련 테이블
  - `faq` - faq 질문들 테이블
  
  

## 3. 페이지별 상세 기획

### 3.1 1페이지: 메인 대시보드 (app.py)

#### 3.1.1 드롭다운 1: 자동차 등록 현황 분석
**기능**: Dual Axis Chart (이중 축 그래프)

- **첫 번째 그래프 (선그래프)**:
  - X축: 2020년~2024년
  - Y축(왼쪽): 전체 자동차 등록대수
  - 데이터 소스: car database 에 environmental_vehicles data table 에 연도별 전체 차량 등록 합계를 사용 

- **두 번째 그래프 (스택형 막대그래프)**:
  - X축: 2020년~2024년
  - Y축(오른쪽): 친환경 자동차 등록대수
  - 스택 구성: 전기차 + 수소차 + 하이브리드
  - 각 차종별 (친환경 자동차 등록대수 대비)비율에 따른 색상 구분, 마우스를 올리면 비율(%)이 함께 표시되도록 설정
  - 연도별 전체 친환경차 등록 규모(막대 높이)와 차종별 구성 비율(스택 색상)을 동시에 파악할 수 있도록 시각화
  - 데이터 소스: car database 에 environmental_vehicles data table 에 구분에 전기차,수소차,하이브리드 데이터 사용
  - 기능 : 각 항목(전기차,수소차,하이브리드) 등을 드롭다운에서 선택하게 된다면 그 데이터를 하이라이트로 시각화

#### 3.1.2 드롭다운 2: 환경 영향 분석
**기능**: Dual Axis Chart + 뉴스 요약판
- **첫 번째 그래프 (선그래프)**:
  - X축: 2020년~2024년
  - Y축(오른쪽): 온실가스 배출량

- **두 번째 그래프 (막대그래프)**:
  - X축: 2020년~2024년
  - Y축(왼쪽): 전체 자동차 대비 친환경 자동차 비율 (%)
  - Y축 최대값: 20%

- **뉴스 요약판**:
  - 내용: "서울시 내연기관 자동차 자동차 출입제한정책"

### 3.2 2페이지: 보조금 정보 (money.py)

#### 3.2.1 드롭다운 1: 공고 현황 분석
**기능**: 스택형 막대그래프 + 표
- **스택형 막대그래프**:
- X축: 연도 (2020년~2024년)
- Y축: 민간공고대수 (전체 막대 높이)
- 스택 구성:
  ▸ 출고대수: 민간공고대수 중 실제 출고된 수량  
     - 마우스를 올리면 출고대수와 민간공고대수 대비 비율(%)이 함께 표시되도록 설정
  ▸ 출고잔여대수: 출고되지 않은 잔여 수량  
     - 마우스를 올리면 잔여대수와 비율(%)이 함께 표시되도록 설정
- 각 연도별 민간공고대수를 기준으로 출고/잔여 수량을 색상으로 시각적으로 구분

- **표**: 2025년 6월 기준 시도별 공고대수 잔여 현황
  - 컬럼: 시도, 민간공고대수, 출고잔여대수

#### 3.2.2 드롭다운 2: 보조금 정보
**기능**: 보조금 표 + TOP5 차량 모델
- **2025년 1대당 지원금 표**:
  - 전기차: 국가보조금, 지자체보조금 (electronic_car 테이블)
  - 수소차: 지원금 (hydrogen_car 테이블)
  - 전기차, 수소차 버튼 클릭하면 해당 표가 화면에 나오도록

- **자동차모델 TOP5**:
  - 컬럼: 순위, 지역,  자동차 모델명
  - 기준: 보조금 지원 금액이 큰 순서
  - 지역을 선택하면 해당지역에서 보조금 지원 금액이 큰 순서대로 5위까지 화면에 나오도록
